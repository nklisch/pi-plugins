tags: [compatibility, security]
---

Qualify the maintained-fork MCP runtime against the real launch boundary to verify that it does not copy or retain plaintext launch values before lease disposal. Portable conformance can prove callback and disposal behavior, but cannot prove non-retention inside a production runtime implementation.
